Screen Definition

In AWE, screens are generated by an XML descriptor and a JSP template. HTML final pages are generated automatically based in those 2 files. To change a screen it's only necessary to update the XML descriptor of the screen, the JSP template doesn't need to be modified.

info

Note: All screen XML descriptors are defined at screen folder. View project structure for more info.
